Transforms

CSS dapat mengatur style dari teks yang ada pada halaman web yang dibuat. Terdapat beberapa property yang dapat digunakan untuku mengatur style teks yaitu sebagai berikut.

  • text align: digunakan untuk mengatur peraturan teks, value yang dapat digunakan yaitu left, center, righ.
  • color: digunakan untuk memberikan warna pada teks.
  • letter-spacing: mengatur jarak antar karakter
  • word-spacing: mengatur jarak antar kota
